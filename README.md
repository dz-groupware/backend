# 조직, 메뉴 관리 솔루션
1. 조직관리: 회사, 부서, 사원의 정보 관리를 관리할 수 있습니다. 
2. 메뉴관리: 메뉴 이름과 구조를 설정할 수 있으며, 적합한 페이지를 설정할 수 있습니다.
3. 권한관리: 사용자의 권한에 따른 메뉴 이용을 설정할 수 있습니다.
